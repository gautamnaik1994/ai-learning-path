# Learning Paths by Goal

This document helps you choose your learning path based on **what you want to build**.  All paths assume you have the **[Prerequisites](../prerequisites.md)** covered.

---

## 1. Build Chatbots & Conversational AI

**Your goal:** Create an AI assistant that can hold conversations, answer questions, or provide support.

**Learning Path:**

1. **Start with basics** → [Generative AI Fundamentals](../gen-ai-learning-path/index.md#beginner-level)
   - Understand transformers and LLMs
   - Learn how language models work
   - **Time:** 1-2 weeks

2. **Learn Prompt Engineering** → [Generative AI - Intermediate](../gen-ai-learning-path/index.md#intermediate-level)
   - Master prompt techniques (few-shot, chain-of-thought, role-playing)
   - Experiment with free LLM APIs
   - **Time:** 1 week

3. **Add Memory & Context** → [Generative AI - Advanced](../gen-ai-learning-path/index.md#advanced-level)
   - Learn LangChain for conversation management
   - Implement memory systems (conversation history)
   - **Time:** 1-2 weeks

4. **Optional: Ground with Real Data** → [RAG (Retrieval-Augmented Generation)](../gen-ai-learning-path/rag.md)
   - Add vector databases to retrieve relevant context
   - Make your chatbot knowledgeable about specific documents
   - **Time:** 1 week

5. **Deploy** → [Production Deployment](../gen-ai-learning-path/production.md) (Coming Soon)
   - Host your chatbot as an API or web service

**Minimal viable project timeline:** 3-4 weeks (skip optional RAG initially)

**Key Skills:** Prompt engineering, LLM APIs, basic LangChain

## 2. Build Recommendation Systems

**Your goal:** Create systems that recommend products, content, or services to users.

**Learning Path:**

1. **Understand ML Fundamentals** → [Machine Learning Basics](../machine-learning/index.md)
   - Supervised learning concepts
   - Model evaluation metrics
   - **Time:** 2 weeks

2. **Learn Key Algorithms** → [Machine Learning - Algorithms](../machine-learning/index.md) (Deep-dive)
   - Collaborative filtering
   - Content-based filtering
   - Similarity measures (cosine similarity, Euclidean distance)
   - **Time:** 1-2 weeks

3. **Optional: Use Embeddings** → [Embeddings & Vectors](../gen-ai-learning-path/index.md#intermediate-level)
   - Semantic similarity using embeddings
   - Vector databases for fast retrieval
   - **Time:** 1 week

4. **Practice on Real Data** → Kaggle datasets or competitions
   - **Time:** 2-3 weeks

5. **Deploy** → [Production Deployment](../gen-ai-learning-path/production.md) (Coming Soon)

**Minimal viable project timeline:** 6-8 weeks

**Key Skills:** Collaborative filtering, similarity metrics, vector databases

---

## 3. Build Computer Vision Applications

**Your goal:** Create systems that understand and process images (object detection, classification, segmentation).

**Learning Path:**

1. **Understand Deep Learning Basics** → [Deep Learning Fundamentals](../deep-learning/index.md#beginner)
   - Core concepts (neurons, activation functions, loss)
   - **Time:** 1 week

2. **Learn Neural Network Fundamentals** → Deep Learning Hands-on Basics
   - Build simple neural networks with PyTorch
   - **Time:** 1 week

3. **Learn CNNs (Convolutional Neural Networks)** → [Deep Learning - CNNs](../deep-learning/index.md#intermediate) (Coming Soon)
   - How convolutions work
   - Architecture designs (AlexNet, ResNet, etc.)
   - **Time:** 2 weeks

4. **Practice on Datasets** → [Deep Learning](../deep-learning/index.md)
   - CIFAR-10, ImageNet, custom datasets
   - **Time:** 2-3 weeks

5. **Advanced: Transfer Learning** → [Deep Learning - Intermediate](../deep-learning/index.md#intermediate)
   - Use pre-trained models for faster results
   - Fine-tune on your specific problem
   - **Time:** 1-2 weeks

6. **Deploy** → [Production Deployment](../gen-ai-learning-path/production.md) (Coming Soon)

**Minimal viable project timeline:** 7-9 weeks

**Key Skills:** CNNs, transfer learning, image preprocessing

## 4. Build Natural Language Processing Applications

**Your goal:** Create systems that understand and generate human language (sentiment analysis, summarization, translation, etc.).

**Learning Path:**

1. **Understand Deep Learning Basics** → [Deep Learning - Beginner](../deep-learning/index.md#beginner)
   - Core concepts
   - **Time:** 1 week

2. **Learn RNNs & Sequences** → [Deep Learning - RNNs/LSTMs](../deep-learning/index.md#intermediate) (Coming Soon)
   - How RNNs process sequences
   - LSTMs and GRUs for long-term dependencies
   - **Time:** 1-2 weeks

3. **Understand Transformers** → [Transformers Introduction](../deep-learning/index.md#intermediate)
   - Attention mechanism
   - Modern transformer architecture
   - **Time:** 1-2 weeks

4. **Use Pre-trained NLP Models** → [Gen AI - Beginner/Intermediate](../gen-ai-learning-path/index.md)
   - Hugging Face Transformers library
   - BERT, GPT models
   - Fine-tuning for your task
   - **Time:** 1-2 weeks

5. **Advanced: Large Language Models** → [Generative AI - Advanced](../gen-ai-learning-path/index.md#advanced-level)
   - Working with LLMs directly
   - Prompt engineering
   - **Time:** 1-2 weeks

6. **Deploy** → [Production Deployment](../gen-ai-learning-path/production.md) (Coming Soon)

**Minimal viable project timeline:** 6-8 weeks

**Key Skills:** Transformers, LLMs, Hugging Face, prompt engineering

## 5. Build Autonomous Agents

**Your goal:** Create intelligent agents that can plan, decide actions, and interact with tools and APIs.

**Learning Path:**

1. **Strong Generative AI Foundation** → [Generative AI - Full Path](../gen-ai-learning-path/index.md)
   - Understand LLMs deeply
   - **Time:** 4-5 weeks

2. **Learn Agent Concepts** → [Agents](../gen-ai-learning-path/agents.md)
   - Agent architectures (single, hierarchical, supervisor)
   - When to use agents vs. RAG
   - **Time:** 1 week

3. **Learn Tool Use & Function Calling** → [Agents - Advanced](../gen-ai-learning-path/agents.md#advanced-level) (Deep-dive)
   - How agents call external tools
   - **Time:** 1 week

4. **Optional: Model Context Protocol** → [MCP](../gen-ai-learning-path/mcp.md)
   - Standardized way to expose tools/data to agents
   - **Time:** 1 week

5. **Build & Test** → Create multi-step agent workflows
   - **Time:** 2-3 weeks

6. **Deploy** → [Production Deployment](../gen-ai-learning-path/production.md) (Coming Soon)

**Minimal viable project timeline:** 8-10 weeks

**Key Skills:** LLMs, agent frameworks (LangGraph), tool use, planning

## 6. Full ML/AI Engineering Career Path

**Your goal:** Become a generalist who can handle any AI/ML task.

**Learning Path:**

This is the most comprehensive path. Follow this sequence:

### **Foundation (4-6 weeks)**

1. [Math Essentials](./math-essentials.md) - Refresher on linear algebra, calculus, probability
2. [Prerequisites](../prerequisites.md) - Python, Jupyter, environment setup
3. [Glossary](./glossary.md) - Reference as you learn

### **Machine Learning (6-8 weeks)**

1. [Machine Learning Fundamentals](../machine-learning/index.md)
2. Practice: 2-3 small projects on Kaggle
3. [Machine Learning - Algorithms Deep-dive](../machine-learning/index.md) (Coming Soon)

### **Deep Learning (8-10 weeks)**

1. [Deep Learning - Beginner](../deep-learning/index.md#beginner)
2. [Deep Learning - Intermediate](../deep-learning/index.md#intermediate)
3. Practice: Image classification and NLP projects
4. [Deep Learning - Advanced](../deep-learning/index.md#advanced) (Coming Soon)

### **Generative AI & Modern Applications (6-8 weeks)**

1. [Generative AI - Complete Path](../gen-ai-learning-path/index.md)
2. [RAG](../gen-ai-learning-path/rag.md)
3. [Agents](../gen-ai-learning-path/agents.md)
4. Practice: Build 2-3 generative AI applications

### **Production & Deployment (2-3 weeks)**

1. [Production Deployment](../gen-ai-learning-path/production.md) (Coming Soon)
2. Learn Docker, FastAPI, cloud deployment
3. Practice: Deploy one model to production

### **Troubleshooting & Debugging (Ongoing)**

- Reference [Common Issues & Solutions](../common-issues.md) as needed

**Total timeline:** 4-5 months

**Key Skills:** Everything! ML fundamentals, deep learning, generative AI, production deployment

## Still Deciding?

**If you want immediate results (1-3 months):**
→ Go for Path #1 (Chatbots) or #2 (Recommendations)

**If you have time to invest (3-5 months):**
→ Follow Path #6 (Full ML/AI Engineering)

**If you want to specialize early:**
→ Pick Path #3 (Computer Vision), #4 (NLP), or #5 (Agents)

**If unsure:**
→ Start with [Machine Learning Basics](../machine-learning/index.md), then decide based on your interest
